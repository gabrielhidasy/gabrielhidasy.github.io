Org Setup

Org capture templates are very flexible and allow for quickly capturing an idea to a note on the fly. Those can easily be used for a blog using ox-hugo, or directly with some more setup.

I didn't want to use ox yet, so here is my config:

(defun get-blog-post-name ()
  "Read post filename and returns it augmented with current date and time"
  (interactive)
  ;; Get title
  (setq post_title (read-string "Title:"))
  (setq post_title_fn (replace-regexp-in-string " " "-" post_title))
  (setq today (format-time-string "%Y-%m-%d-%h-%M"))
  (setq post_filename  (concat "~/blog/content/post/" post_title_fn "-" today ".org"))
  (find-file post_filename))

(setf org-capture-templates
      '(("B" "Blog Post" plain (function get-blog-post-name) 
        (file "~/.emacs.d/template_hugo_post.org"))))

And here is the template:

---
layout:     post
title:      "%(concat post_title)"
subtitle:   "%^{Subtitle}"
date:       %<%Y-%m-%e>
author:     "gabrielhidasy"
image:      "img/post-bg-2015.jpg"
showtoc:    "false"
tags:
    - %^{Tag}
draft:      "true"
categories: ["Tech"]
---
* %?

With this configutation, anc my default C-c c template, C-c c B starts a post, requests for title, subtitle, auto-fills date and saves the post in its own file in the right place.
